## 💁 서비스 소개



> 가족만의 글씨체로 추억과 일상을 공유하는 디지털 가족 앨범 서비스

<br />


### 서비스 설명 (주요 기능)

1. 나만의 손글씨 : Few-shot Font Generation 모델 AI를 활용해 나의 손글씨를 폰트로 생성
2. 추억 갤러리 : 추억이 담긴 사진을 업로드 할 수 있는 공간
    - 사진 뒷면 엽서에 텍스트나 음성으로 감상평 작성
    - 텍스트의 경우 만들어진 폰트가 있다면 폰트 적용
3. 편지: 텍스트나 음성으로 편지 작성
    - 느린 편지로 개봉 날짜 지정 가능
    - 텍스트의 경우 만들어진 폰트가 있다면 폰트 적용
4. 방울 캘린더 : 가족 일정 등록 가능, 앨범과 연결
